We can see that the tree first split on number of times 90 days late. For those for whom this was true that it was <.5, revolving utilization was the next most significant predictor, followed by the number of times 30-59 days past due. For simplicity I implemented a max depth of 4.

For those for whom they were later >.5 time, the next most important criteria was different -- this time it was zip code. We can trace down the paths to classify individuals.
